# Unity GGemCo TCG 패키지
유니티에서 2D TCG 게임을 만들기 위한 라이브러리 입니다.

## 기능
- **TCG 전투 시스템 완성**: 카드 명령 → 연출 → 결과까지 이어지는 전투 흐름 확립, 종료 조건·AI 대전·셔플 전략 등 핵심 전투 로직 구현
- **Ability 시스템 전면 개편**: 카드 타입 기반 Ability 구조 통합, Permanent·Equipment 등 확장 카드 타입 지원 및 UI 연출 파이프라인 구축
- **영웅·덱·콜렉션 핵심 플레이 루프 구축**: 영웅 카드 시스템, 디폴트 덱/대결 관리, 카드 콜렉션 UI까지 TCG 핵심 플레이 사이클 완성
- **TCG 패키지 자동화 및 사용성 확보**: 자동 기본 설정 툴, 샘플 데이터·리소스·Localization 포함으로 즉시 사용 가능한 TCG 패키지 제공
